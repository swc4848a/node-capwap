#node-capwap

### CAPWAP implementation in JavaScript (Work in progress)

Control And Provisioning of Wireless Access Points (CAPWAP) Protocol implementation for Node.js
written in JavaScript.

### Objectives

### Running

Install Dependent using NPM.

```
npm install
```

Start

```
node app.js
```
